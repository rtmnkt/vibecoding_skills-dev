# .docs/specs/ — エージェント向け仕様メモ

このフォルダはAIエージェントが開発中に得た知識・判断・ハマりポイントを記録する場所。
ファイル番号は `.docs/requirements/` と対応する。

## ルール

- **読者はエージェント自身**。人間への丁寧さ不要。コンテキスト効率を最優先。
- front matter に `requirement`, `commits`, `files` を記載。検索性のため。
- 本文はキーバリュー的・箇条書き的に書く。散文禁止。
- 「なぜその判断をしたか」「何にハマったか」「次回変更時に知るべきこと」を優先記録。
- AA やコードブロックなど冗長な内容はファイルパスで参照し、ここには書かない。
- この 0000.md 自体は人間が読む唯一のファイル。他は全てエージェント最適化。
